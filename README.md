【游戏名称】
过马路小游戏

【游戏介绍】
这是一款H5原生开发的3D休闲闯关类小游戏，灵感来自经典游戏《Crossy Road》。游戏中，你控制一只小鸡穿越繁忙的马路和森林，躲避来往的汽车和卡车。！

【玩法说明】
游戏中，小鸡需要不断向前穿越不同的地形，包括草地、马路和森林。

障碍：
1. 马路上有汽车和卡车不断来回行驶，速度和方向各不相同。
2. 森林中有树木阻挡，无法穿过树木所在的位置。
3. 如果被车辆撞到，游戏立即结束。
4. 不能移动到地图边界之外。

操作：
点击屏幕上方向按钮（或使用键盘方向键）：
↑ 键 - 向前移动（穿越马路）
↓ 键 - 向后移动
← 键 - 向左移动
→ 键 - 向右移动

目标：
尽可能向前移动更多步数，每向前移动一步得1分。躲避所有车辆，挑战最高分！


【技术介绍】
HTML5原生开发，主要语言是JavaScript。
